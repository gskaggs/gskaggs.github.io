This project's purpose is to measure pupil size given infared images of the eye. The code is planned to be used in research conducted at the UT Austin Dell Medical School for the development of novel theraputic treatments for patients with Multiple Sclerosis. 

The principal investigator for this project is [Thomas Varkey, MBA,](https://www.linkedin.com/in/thomas-c-varkey/) a medical student and researcher at Dell Medical School. Feel free to browse my code at [its public GitHub repository.](https://github.com/gskaggs/Pupilometer-Analysis)
