Here are a few of my projects I'm most proud of.
Pyxeled

Transforming normal photographs into aesthetic pixel art. That's Pyxeled. Programmed in Python and inspired by the paper Pixelated Image Abstraction by Gerstner et al., this application leverages intelligent clustering algorithms to optimally render images at lower resolutions and limited color palettes.

To learn more about the algorithms behind Pyxeled and to see more art generated using it, check out its page on my website. Additionally, you can browse all the code for this project and find instructions on how to run the program on your own images here.

The Great Sand Dunes

This was my final project in the computer graphics course I took last semester. Working in a team of two, I wrote C++ code to simulate the physics of sand particles to render desert geometries. You can find the code for this project, an in-depth explanation of the physics and algorithms we used, and instructions on how to run it on your own machine on its public GitHub page.

A Genetic Algorithm for the Iterated Prisoner's Dilemma

After reading Richard Dawkin's The Selfish Gene, I was fascinated by the evolution of cooperation and altruism and started exploring a game theoretic phenomenon called the iterated prisoner's dilemma. One experiment I programmed in Python was a genetic approach to the problem. I wrote about this project in my blog and even wrote some Javascript so readers could compete against the winning strategy produced by the genetic algorithm. You can find the blog post and interactive simulation here.

Pupilometer

In this project I worked with a team of researchers at UT's Dell Medical School to develop a new therapeutic device for MS patients. My contribution was to program a means of measuring pupil size in infrared images of the human eye. My solution was to borrow an algorithm from graph theory called "Breadth First Search" to determine the width of the eye in pixels. You can find the code for this project on its public GitHub repository.

Plexus

Plexus was a side-scrolling adventure RPG I created in highschool with a team of three other students. We submitted the game to the Business Professionals of America and placed among the top 15 in their national Software Engineering Team competition. If you'd like, you can download the Java code on the project's GitHub page and play it on your own machine.

Ray Tracer / Web Crawler

Ray Tracer and Web Crawler were two course projects I worked on during my time at UT. Although I feature them on my resume, I can't include code for either project since every year UT students have to do these projects in Computer Graphics and Data Structures, respectively.
